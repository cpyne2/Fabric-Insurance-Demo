# Fabric-Insurance-Demo

This project is a Microsoft Fabric and Power BI insurance claims demo environment built to showcase the platform's end-to-end analytics and AI capabilities. It consists of a SQL database with three core tables — Customers, Policies, and Claims — where roughly 10% of claims are intentionally filed outside the policy coverage window to demonstrate a real-world claims denial scenario, supported by a Power BI semantic model with DAX measures that surface key KPIs like denial rate, approval rate, and invalid claims exposure. A Fabric Lakehouse table of adjuster notes adds a second data source that provides free-text investigative narrative for each claim, enabling richer cross-source analysis. A Fabric Data Agent called the Claims Data Agent sits on top of both sources, allowing non-technical users to investigate claims, identify payment risks, and retrieve adjuster investigation history using plain English questions — with no SQL, DAX, or reporting skills required.
